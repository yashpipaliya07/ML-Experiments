# ML Experiments

This repository contains experiments and implementations of fundamental **Machine Learning algorithms** using Python.
The goal of this repository is to understand ML concepts by implementing algorithms and optimization techniques step-by-step.

## 📁 Repository Structure

```
ML-Experiments
│
├── regression
│   ├── linear_regression_gradient_descent.ipynb
│   └── regularization_l1_l2_elasticnet.ipynb
│
├── datasets
│   ├── train.csv
│   ├── test.csv
│   └── submission_example.csv
│
└── README.md
```

## 🚀 Implemented Concepts

* Linear Regression
* Batch Gradient Descent
* Stochastic Gradient Descent
* Mini-Batch Gradient Descent
* L1 Regularization (Lasso)
* L2 Regularization (Ridge)
* Elastic Net Regularization

## 🛠️ Libraries Used

* Python
* NumPy
* Pandas
* Matplotlib (if used)

## 🎯 Purpose

This repository is created to experiment with machine learning concepts and build a strong understanding of algorithm implementations.

More ML algorithms and experiments will be added in the future.
